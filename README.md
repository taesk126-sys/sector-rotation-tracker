# US Sector & Theme Rotation Tracker

Dashboard ติดตาม sector rotation ของตลาดหุ้นสหรัฐ อัปเดตอัตโนมัติทุกเช้า (หลังตลาดปิด) ผ่าน GitHub Actions

- ข้อมูล: Yahoo Finance (ราคาปิดรายวัน)
- 25 ธีมย่อย (equal-weight) + 11 Sector ETF
- Relative Rotation Graph, Heatmap, Momentum Playbook, Breadth & Volume, ETF Appendix

เครื่องมือประกอบการวิเคราะห์ ไม่ใช่คำแนะนำการลงทุน
